# dedeme_lib

Crystal utilities for personal use.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  dedeme_lib:
    github: dedeme/dedeme_lib
```

## Usage

```crystal
require "dedeme_lib"
```
